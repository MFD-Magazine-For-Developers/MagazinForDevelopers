Java를 시작하기 전에 프로그램 소스를 작성하는 툴, 소스를 컴파일하는 프로그램 등을 설치해야 합니다. 
Java 프로그래밍을 하기 위해 필요한 일은 JDK(Java Development Kit)를 설치하는 것입니다.
JDK를 설치하면 javac라는 컴파일러가 제공됩니다. 
그러나 통합 개발 환경은 제공해 주지 않기 때문에 별도의 개발용 프로그램을 써야 하며 
대표적으로 [이클립스],[넷빈즈],[IntelliJ IDEA] 등이 있습니다.

실전 Java 개발을 할 때 프로젝트에 필요한 라이브러리 관리나 프로젝트 결과물 배포 등 
프로젝트 빌드 관리를 위한 도구로는 Ant, Maven, Gradle이 있습니다.
